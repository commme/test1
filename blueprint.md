# Career Decision Psychological Test Project

## Overview
A fun, interactive 7-question psychological test designed to help users decide their career path: "Stay in Job", "Quit & Start Business", or "Freelance/Other". The application uses a playful, RPG-style interface to engage users.

## Project Architecture
- **Frontend**: HTML5, CSS3 (Modern features like variables, flexbox/grid, animations), JavaScript (ES6+).
- **No Frameworks**: Pure Vanilla JS as per project guidelines.
- **Responsiveness**: Mobile-first design.

## Current Plan: Initial Implementation
1.  **Content Design**: Create 7 scenario-based questions with weighted answers.
2.  **UI Construction**:
    -   Landing screen with a "Start" button.
    -   Quiz container for displaying questions one by one.
    -   Result screen with a fun description and "Restart" button.
3.  **Logic Implementation**:
    -   `main.js` to handle navigation, score tracking, and result calculation.
4.  **Styling**:
    -   `style.css` for a vibrant, engaging look using modern CSS.

## Features
-   **7 Questions**: Carefully crafted to gauge risk tolerance, independence, and creativity.
-   **Dynamic Results**:
    -   *The Loyal Guardian* (Keep the job)
    -   *The Daring Pioneer* (Start a business)
    -   *The Free Spirit* (Freelancer/Digital Nomad)
-   **Progress Bar**: Visual indicator of quiz progress.
