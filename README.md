# Career Decision Psychological Test (대창업의 시대)

This is a fun, interactive web-based psychological test to help users decide their career path: **Stay in Job**, **Start a Business**, or **Become a Freelancer**.

## Features
- **7 Engaging Scenarios**: Situational questions to determine your work style.
- **Dynamic Scoring**: Tracks user choices across three categories.
- **Animated UI**: Smooth transitions and modern design.
- **Responsive**: Works on desktop and mobile.

## Technologies Used
- HTML5
- CSS3 (Animations, Flexbox)
- Vanilla JavaScript (ES6 Modules)

## How to Run
Simply open `index.html` in a browser or use a live server.
